🟡 About Pac-Man

Pac-Man is a classic arcade game where you control a yellow character in a maze. The goal is to eat all the dots while avoiding ghosts.

🎮 How to Play

Move Pac-Man using arrow keys. Eat all dots to win. Avoid ghosts or you lose. Eat power pellets to chase and eat ghosts for a short time.
